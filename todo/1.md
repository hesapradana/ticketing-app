a. Ruang Lingkup Capstone Project
Mahasiswa wajib menyelesaikan Modul 1 hingga Modul 11 yang mencakup:
Link Notion : https://bit.ly/NotionBimkar
1. Manajemen Event & Kategori: Admin dapat melakukan CRUD
kategori dan event secara menyeluruh.
2. Manajemen Tiket (Kuota):
●
Admin menetapkan kuota tiket per event.
●
Sistem secara otomatis mengurangi kuota tiket setiap kali user
melakukan pemesanan (tanpa pembayaran/langsung tercatat).
3. Handling Kuota Habis:
●
Sistem harus memberikan validasi dan error handling/notifikasi
ketika kuota tiket suatu event sudah habis (Sold Out).
●
Nilai Plus: Menampilkan indikator visual jika tiket menipis (misal:
sisa < 5).
4. Riwayat Transaksi: Menampilkan log transaksi pemesanan tiket pada
sisi Admin dan User.
